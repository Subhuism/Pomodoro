# ⚡ Pomodoro Protocol

A modern Neo-Brutalist Pomodoro timer built for students, developers, creators, and deep-work enthusiasts.

## Features ##

### Custom Protocols
Create your own productivity system:
- tasks are Stored in Local Storage , reload with no fear
- Session completion % shown aka live session counter
- Auto-start next session

### Task Management
- Create tasks
- Track progress alongside focus sessions
- Stay organized while working

### Theme Switching
Choose from multiple hand-picked color themes:
Explore it Yourself 😉

**One click instantly updates the application's visual style.**

### Responsive Design
Built to work across:

- Mobile
- Tablet
- Desktop

## Why Pomodoro Protocol?

Most Pomodoro apps only provide a timer.

Pomodoro Protocol helps you:
- Build structured work sessions
- Manage tasks
- Track progress visually
- Customize your entire productivity workflow

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/pomodoro-protocol.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
